# okr
A stateless cli for managing [OKRs](https://en.wikipedia.org/wiki/OKR). Accepts `json` as input for now. See `/example` for json structure.

# install
```bash
$ npm i okr -g
```

# usage
```bash
# list
$ okr -f <file>
```
